# Basic React Project (iPhone17)

Português / English

## Português

**Sobre:**

Projeto front-end simples criado com React e Vite. É uma base mínima para demonstrar componentes React, estilização com CSS e estrutura de pastas organizada para um pequeno site/aplicação.

**Tecnologias:**

- **React** (JSX)
- **Vite** (bundler / dev server com HMR)
- **CSS** (arquivo `index.css`)
- **ESLint** (configuração básica)
- Estrutura de componentes em `src/components`

**Como executar (desenvolvimento):**

```bash
npm install
npm run dev
```

**Observações:**

- Projeto preparado para desenvolvimento local com Vite. Personalize componentes e estilos conforme necessário.

## English

**About:**

Simple front-end project built with React and Vite. This repository is a minimal starter demonstrating React components, CSS styling, and a small organized folder structure for a demo site/app.

**Technologies:**

- **React** (JSX)
- **Vite** (dev server / bundler with HMR)
- **CSS** (`index.css`)
- **ESLint** (basic configuration)
- Components located in `src/components`

**How to run (development):**

```bash
npm install
npm run dev
```

**Main structure:**

- `index.html` — entry HTML file
- `src/main.jsx` — React entry point
- `src/App.jsx` — main app component
- `src/index.css` — global styles
- `src/components/` — reusable components (Colors, Hero, Highlights, Intro, NavBar)
- `public/img/` — public images used by the project

**Notes:**

- Ready for local development with Vite. Customize components and styles as needed.
